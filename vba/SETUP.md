# VBAセットアップ手順

## 1. 事前準備
1. `○在庫帳○27期_テスト用少データ.xlsx` を **名前を付けて保存** → `.xlsm` 形式で保存
2. Excel の `ファイル` → `オプション` → `トラスト センター` → `トラスト センターの設定` → `マクロの設定`
   - **「VBAプロジェクト オブジェクト モデルへのアクセスを信頼する」にチェック**
3. `Alt + F11` でVBAエディターを開く

---

## 2. mSetup をインポート
1. VBAエディター: `ファイル` → `ファイルのインポート`
2. `mSetup.bas` だけをインポート

---

## 3. 自動生成を実行
1. `Alt + F8` → `Setup` を選択 → 実行
2. 「セットアップ完了！」が表示されれば成功
   - `mMain` / `frmSettings` / `frmInput` が自動生成されます

---

## 4. マクロ実行
- `Alt + F8` → `Main` を選択して実行
- またはシートにボタンを配置して `Main` を割り当て

---

## テスト用画像フォルダ
任意のフォルダにJPG/PNG/BMPファイルを数枚置いてテストしてください。
処理後は同フォルダ内に `処理済み/` フォルダが自動作成され、画像が移動します。

---

## トラブルシュート
- **「Designer 取得失敗」エラー**
  → 手順1-2 の信頼設定を再確認。設定後 Excel を再起動。
- **「プログラムによる Visual Basic プロジェクトへのアクセスは信頼性に欠けます」**
  → 同上。信頼設定後に再実行。
