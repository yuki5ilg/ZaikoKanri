# VBAセットアップ手順

## 準備
1. `○在庫帳○27期_テスト用少データ.xlsx` を **名前を付けて保存** → `.xlsm` 形式で保存
2. Excel の `ファイル` → `オプション` → `トラスト センター` → `トラスト センターの設定` → `マクロの設定`
   - **「VBAプロジェクト オブジェクト モデルへのアクセスを信頼する」にチェック**
3. Excel を再起動
4. `Alt + F11` でVBAエディターを開く

---

## セットアップ（1回だけ）

1. VBAエディター: `ファイル` → `ファイルのインポート`
2. **`mSetup.bas` だけ**インポート（他のファイルは不要）
3. `Alt + F8` → `Setup` を実行
4. 「セットアップ完了！」が表示されればOK

> `mMain`・`frmInput` が自動生成されます。

---

## 実行方法

- `Alt + F8` → `Main` を実行
- またはシートにボタンを配置して `Main` を割り当て

---

## 使い方

1. フォームが開いたらシート名とフォルダを指定して **「開始」**
2. 画像が表示されたらデータを確認・修正して **「取込」**
3. 次の画像が自動で表示される（全件完了で自動終了）
4. 売上情報は下部の「売上」セクションに入力すれば同時に書き込まれる

---

## 注意

- `frmInput.bas` / `mMain.bas` は開発用ファイルなので**インポート不要**
- 画像処理後は同フォルダ内に `処理済み/` フォルダが自動作成され画像が移動する
